Learning GitHub Projects

Step 1: Create a New Repository

  1. Go to GitHub and log in.
  2. Click on the "+" icon in the upper-right corner and select "New repository."
  3. Name your repository (e.g., learning-github-projects).
  4. Add a description if you like.
  5. Choose "Public" or "Private" depending on your preference.
  6. Check "Initialize this repository with a README."
  7. Click "Create repository."

Step 2: Create a GitHub Project

  1. Navigate to your new repository.
  2. Click on the "Projects" tab.
  3. Click "Create a project."
  4. Name your project (e.g., Project Learning).
  5. Optionally, add a description.
  6. Click "Create project."

Step 3: Set Up Your Project Board

  1. After creating the project, you'll see a kanban-style board.
  2. By default, you'll have three columns: "To do," "In progress," and "Done."
  3. You can add more columns by clicking the "+ Add column" button.

Step 4: Add Issues to Your Project

  1. Navigate to the "Issues" tab of your repository.
  2. Click "New issue."
  3. Title your issue (e.g., Set up project structure).
  4. Add a description if needed.
  5. Click "Submit new issue."
  6. Repeat to create more issues (e.g., Create initial files, Set up CI/CD, etc.).

Step 5: Add Issues to Your Project Board

  1. Go back to your project board.
  2. Click on "+ Add cards" in the "To do" column.
  3. You'll see a list of issues you created. Drag them into the "To do" column.

Step 6: Move Issues Across Columns

  1. As you work on tasks, you can drag issues from "To do" to "In progress" and then to "Done" as they are completed.

Setting Up Project Structure

mkdir src
touch src/index.html src/style.css src/app.js
npm init -y

Creating Initial Files

<!DOCTYPE html>
<html lang="en">

    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">

<body>
    <h1>Welcome to GitHub Projects</h1>
    <script src="app.js"></script>
</body>
</html>

style.css

body {
    font-family: Arial, sans-serif;
    text-align: center;
}

app.js

console.log("Hello, GitHub Projects!");

GitHub Actions Workflow (CI/CD)

Create a .github/workflows/ci.yml file:

name: CI

on: [push, pull_request]

jobs:
  build:
    runs-on: ubuntu-latest

    steps:
    - uses: actions/checkout@v2
    - name: Set up Node.js
      uses: actions/setup-node@v2
      with:
        node-version: '14'
    - run: npm install
    - run: npm test

Deploy to GitHub Pages

  1. In your repository, go to Settings > Pages.
  2. Select the branch you want to deploy from (e.g., main).
  3. Click "Save."
